# Character Creator (Python)

This is a small learning project built while studying Python.

## Features
- Validates character name
- Validates stats using constraints
- Displays character stats using visual bars
- Handles edge cases required by strict tests

## What I learned
- Function design in Python
- Input validation with isinstance
- Handling constraints and order of checks
- Importance of exact output formatting
